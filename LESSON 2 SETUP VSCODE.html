SETUP VSCODE VÀ KHỞI TẠO DỰ ÁN ĐẦU TIÊN

1. Khởi tạo dự án bằng VSCODE

B1: Mở visual studio code
B2: Tạo thư mục lưu trữ
B3: Mở thư mục đã tạo
B4: Tạo dự án

Click

2. Tạo webpage đầu tiên

B1: Nhấn Shift + 1 (!) để tạo ra dòng lệnh mặc định của file HTML
B2: Đổi <html lang="en> sang <html lang="vi">
B3: Đổi tên tiêu đề web mục <title>Document</title>
B4: Điều chỉnh font chữ Vào Settings (Ctrl + ,) -> font -> đều chỉnh ở font size

3. Cài đặt Extension

a. Prettier : tự động định dạng lại code trông đẹp và dễ đọc hơn

Cài đặt xong -> Vào Settings (Ctrl + ,) -> default format -> ở default formatter chọn prettier - code fomatter

b. Format on save

Settings (Ctrl + ,) -> tìm kiếm "format on save" -> ở mục format on save tích chọn
Cài đặt auto save: -> tìm kiếm "auto save" -> ở mục auto save chọn "onFocuschange"

c. Material icon theme

Hiển thị icon các loại tệp

d. Image preview

Hiển thị ảnh preview

e. Color highlight

ứng dụng trong CSS

f. Auto rename tag

tự động rename tag khi thay đôi tag

g. liver server

tự động update page ontime khi lưu file

h. Live server preview

sau khi cài đặt nhấn "F1" chọn "Show live server preview"

i. code spell check

Tự động check chính tả

k. HTML tag wrap

Chèn thêm thẻ vào khối sẵn có sử dụng tổ hợp phím "ATL + W"

l. HTML to CSS autocompletion

tự động thêm gợi ý tên class, id trong CSS
