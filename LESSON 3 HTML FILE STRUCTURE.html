1. Cấu trúc cơ bản của file HTML

          
              <!DOCTYPE html> 

              <html lang="vi"> 

              <head> 

              <meta charset="UTF-8" /> 

              <meta name="viewport" content="width=device-width,
              initial-scale=1.0" /> 

              <title>Cấu trúc file HTML</title> 

              </head> 

              <body>
1. <!DOCTYPE html>
Khai báo doctype (document type declaration)
2. <html lang="vi"> </html>
Thẻ HTML định nghĩa phần thân của tải liệu html, sử dụng thêm thuộc tính lang="vi" để chỉ ngôn ngữ được viết của website
3. <head> </head>
Chứa các thông tin về trang web mà không hiển thị trực tiếp
Các thông tin trong thẻ head như: Tiêu đề trang <title>Cấu trúc file HTML</title>, mô tả trang, từ khóa, thông tin tác giả, các tập tin css và javascript được liên kết với trang.
4. <body> </body>
Chứa nội dung chính của trang web Các phần tử HTML bên trong thẻ body hiển thị trực tiếp trên trình duyệt của người dùng, và tạo thành nội dung chính của trang web.

f. Thẻ ul & ol

thẻ ul: tạo list không theo thứ tự lệnh ul>li*3

  • nội dung chính
  • nội dung phụ
  • ví dụ

thẻ ol: tạo list theo thứ tự lệnh ol>li*3

  1. nội dung chính
  2. nội dung phụ
  3. ví dụ

Lênh ol có thể thay đổi định dạng đánh số <ol type=""></ol>

Các lệnh ul & ol có thể lồng nhau:

ol>li*2>ul>li*3
  1. mục 1
    • nd 1
    • nd 2
    • nd 3
  2. mục 2
    • nd 4
    • nd 5
    • nd 6
