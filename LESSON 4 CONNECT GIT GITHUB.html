Hình ảnh cấu trúc trao đổi dữ liệu

1. DEPLOY WEBSITE SỬ DỤNG VSCODE

  1. Tạo Repository trên GITHUB để lưu trữ dự án trên máy chủ GIT
    New repository -> repository name -> thực hiện tập hợp các dòng lệnh trên VSCODE 
                
    1. create new repository on the command line
    git init
    git add .
    git commit -m "nội dung"
    git branch -M main
    git remote add origin https://github.com/.......(có link ở repository)
    git push -u origin main
    2. push an existing repository from the command line
    git remote add origin https://github.com/.......(có link ở repository)
    git branch -M main
    git push -u origin main
    3. Các lệnh nâng cao
    git branch: xem các nhanh local
    git branch -a: xem các nhanh cả local và remote
    git pull : để kéo các lệnh sửa đổi từ repository về local
    git branch -c "tên nhanh": copy nội dung nhánh hiện tại sang nhánh mới
    git checkout "tên nhánh": chuyển sang làm việc nhánh khác
    git checkout -b "tên nhánh": copy toàn bộ code nhánh hiện tại sang nhánh mới và con trỏ sang nhánh mới
    git branch -d "tên nhánh": xóa nhánh tại local
    git push -d origin "tên nhánh": xóa nhánh tại remote repository
    git log & git log --online : kiểm tra lịch sử code
  2. Tạo dự án trên máy tính cá nhân: dùng để code và lưu trữ cục bộ trên máy tính
  3. Khởi tạo GIT
    1. Mở terminal
    2. Config mặc định chọn git bash khi mở terminal (mục select default profile)
    3. Kiểm tra trạng thái bằng lệnh git status
    4. Khởi tạo một kho lưu trữ mới trên local
      git init
  4. add thư mục dự án vào staging area chờ commit
  5. Commit và đẩy code lên GITHUB
  6. Deploy website
    vào repository đã tạo -> Settings -> Pages -> branch (chuyển sang main) -> Vào action -> Chờ deploy website
