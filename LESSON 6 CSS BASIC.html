1. Ba vị trí đặt CSS


Độ ưu tiên các vị trí đặt CSS: CSS inline ưu tiên đầu tiên, đối với CSS Internal và CSS External, cái nào khai báo sau sẽ ghi đè lên cái trước đó

a. Inline

CSS nội tuyến: Viết ngay bên trong phần tử của HTML

  • Khó thay đổi, mổi lần thay đổi phải thay đổi riêng lẻ
  • Không tái sử dụng được
  • Khó quản lý
  • Khó tách biệt cấu trúc

b. Internal

CSS nội bộ: là cách viết CSS ngày trong cặp Style (đặt trong khối HEAD)

  • Ưu điểm:
    • Code tách biệt với phần tử
    • có thể áp dụng với toàn trang
  • Nhược điểm
    • Vẫn khó quản lý khi dự án lớn
    • Quá nhiều mã CSS làm HTML trở nên cồng kềnh

c. External

CSS bên ngoài: Trong trường hợp này sẽ áp dụng cho toàn bộ file HTML nằm trên trang web, tránh việc phải copy CSS ra tất cả các file

Tạo file CSS -> khai báo trong tệp HTML để link đến tệp CSS -> rel="Tên file CSS" thông báo đến tệp HTML về tệp CSS áp dụng

2. Các thuộc tính cơ bản

Tra cứu các thuộc tính cơ bản
Hiện ảnh các thuộc tính

a. Opacity, display và visibility

1.1 Opacity Quy định độ trong suốt

h2 {opacity: 100% - 0% (điều chỉnh độ trong suốt)}

Đặc điểm:

  • Không nhìn thấy trên giao diện
  • Vẫn chiếm không gian
  • Vẫn tương tác được

1.2 display: none Ẩn phần tử khỏi giao diện (xóa cả không gian nó đang chiếm)

h2 {display: none;}

1.3 visibility: Ẩn hoặc hiện element

h2 {visibility: hidden/ visible}
  • Không nhìn thấy trên giao diện
  • Vẫn chiếm không gian
  • Không tương tác được
So sánh opacity - visibility - display

b. Favicon

Link đến danh mục các CSS selectors

Kiểm tra lỗi của code

Để tác đông lên nhiều phần tử có thể sử dụng class thay thế id

ở CSS để goi ID đánh "#", để gọi class đánh "."

Tác động lên toàn bộ file HTML: * {} (dùng thẻ *)

element.class:Tác động lên các thẻ có class không tác động toàn bộ như thẻ *

Có thẻ liệt kê các phần tử để tác động ở CSS bằng ngăn cách nhau bởi các dấu phẩy

      
        # id {} (gọi ID)
        . class (gọi class)
        * {} (Tác động lên toàn bộ thẻ HTML)
        element.class {} (Tác động lên từng element)
        h1, h2, ul {
          color: red, 
          font size: 20px
        } 
        (Tác động lên các element được chọn)
Space selectors & >selector & +selector & ~selector 

        div p {
          color: blue,
          font size: 20px,
        } (Tắc động đến toàn bộ thẻ con p bên trong div)
        div>p {
          color: red,
          font style: italic,
        } Tác động đến các thẻ con P trực tiếp bên trong thẻ div, không tác động lên các thẻ con gián tiếp
        div + p {tác động lên thẻ p đặt ngay sau thẻ div}
        div ~ p {tác động tất cả các phần tử P đặt sau thẻ div}
        đối với attribute selector 
        [attribute] -> tác động lên các phần tử có attribute ngày
        [attribute]="value" -> chọn các phần tử có giá trị bằng "value"
        [attribute]*="value" -> chọn tất cả các phần tử có chưa "value" trong giá trị
        [attribute]~= "value" -> tác động lên phần tử có thuộc tích trùng với value
        [attribute] |="value" -> tác động lên các phàn tử có thuộc tính bắt đầu bằng "value"
        [attribute]^="value" -> tác đông lên phần tử có giá trì "value" 
        [attribute]$="value" -> tác động các phần tử kết thúc bằng "value"
